傻瓜流程

  1. 打開 terminal
  2. $ cd ~/Projects/eCourt  #這邊假設你的 workingspace 在 ~/Projects/eCourt
  3. $ git pull 
  4. 修改你要改的部分
  5. 測試新功能或問題有沒有被解決
  6. $ git add file1 file2 ... # file1, file2 ... 表示新加入的檔案或是有被改過的檔案
  7. $ git commit -m "message" # message 是一句簡述這次改了些什麼
  8. $ git push
  9. 部署到 heroku
  • 好branch，不用嗎？
  • 這是傻瓜流程，先不管 branch
